# DoS / DDoS Detection

Machine learning-based detection of DoS and DDoS attacks using the CIC-IDS2018 dataset.

## Setup

1. Download the CIC-IDS2018 DoS/DDoS `.parquet` files and place them in the `datasets/` folder.
   See `datasets/readme.md` for details.
2. Open `ML.ipynb` and run the notebook.
